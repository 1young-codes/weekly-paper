# Week 01

## Q1. 선형대수학이란 어떤 학문이며, 왜 머신러닝에 필요한가?

**한 줄 요약:** 선형대수학은 다차원 공간에서 데이터의 변환과 방향성을 다루는 학문이며, 머신러닝에서 수억 개의 데이터를 하나의 수식으로 묶어 초고속으로 계산하기 위해 반드시 필요하다.

### 선형대수학이란

직선이나 평면 같은 '선형적인' 관계를 다차원 공간(벡터와 행렬)으로 확장하여 연구하는 수학의 한 분야이다.

### 왜 머신러닝에 필요한가

- **데이터의 표현:** 하나의 데이터(이미지의 픽셀 값, 사용자의 구매 기록 등)는 수많은 특징(Feature)을 가진 **벡터**로 표현된다. 이 벡터들이 모인 데이터셋이 바로 **행렬**이다.
- **공간의 변환:** 머신러닝 모델의 학습은 결국 데이터를 가장 잘 분류·예측하도록 공간을 비틀고 회전시키는 과정(**선형 변환**)이다.
- **연산의 효율성:** 수천만 개의 데이터를 하나씩 연산하면 너무 느리다. 거대한 행렬 연산으로 바꾸면 GPU로 병렬 계산을 끝낼 수 있다.

### 예시: 엑셀 시트 통째로 계산하기

사과 10개, 바나나 5개의 가격을 하나씩 더할 수도 있지만, `[10, 5]` 벡터와 `[사과가격, 바나나가격]` 벡터를 한 번에 곱하는 것이 선형대수학의 방식이다. 복잡한 데이터 세상을 엑셀 표(행렬) 하나로 묶어 컴퓨터가 한 번에 처리하게 만드는 '치트키' 같은 학문이다.


## Q2. EDA(Exploratory Data Analysis)란 무엇인가?

**한 줄 요약:** EDA는 본격적인 모델링에 앞서 데이터의 분포, 통계치, 시각화 자료를 통해 데이터 자체의 숨겨진 특징과 패턴을 탐색하고 직관을 얻는 과정이다. 

### EDA란

통계학자 존 튜키(John Tukey)가 제안한 방법론으로, 가설을 세우고 검증하기 전에 데이터가 우리에게 무엇을 말하려 하는지를 먼저 들어보는 단계이다.

### 어떻게 진행하는가?

평균, 중앙값, 표준편차 같은 기술 통계량을 확인하고, 히스토그램, 산점도(Scatter Plot), 박스 플롯(Box Plot) 같은 시각화 도구를 사용해 데이터의 모양과 변수 간의 상관관계를 파악한다.

### 왜 중요한가?

데이터에 숨겨진 오류(이상치, 결측치)를 잡아내고, 어떤 머신러닝 알고리즘이 적합할지, 어떤 특징(Feature)을 모델에 넣어야 할지 결정하는 나침반 역할을 하기 때문이다.

### 비유: 의사가 본격 치료 전에 검사부터 하는 것. 

바로 수술(모델링)에 들어가지 않고 먼저 엑스레이 찍고 피 뽑고 혈압 재면서(기술 통계, 시각화) 환자 상태를 파악한다. 
그러다 예상 못 한 수치(이상치)를 발견하면 치료 방향이 바꾼다. EDA(Exploratory Data Analysis)는 "데이터의 건강검진"이다.


## Q3. 결측값 · 중복값 · 이상치 처리 방법

**한 줄 요약:** 데이터 전처리는 쓰레기를 넣으면 쓰레기가 나오는(Garbage In, Garbage Out) 머신러닝의 문제를 방지하기 위해, 오염되거나 빠진 데이터를 지우고 채워 넣어 데이터의 품질을 높이는 작업이다.

### 자세한 설명

1. **결측값 (Missing Value):** 데이터 수집 누락이나 오류로 인해 값이 비어 있는 상태 (Null / NaN)
   - 삭제: 결측치가 너무 많다면 행이나 열을 삭제함.
   - 대치(Imputation): 평균값, 중앙값, 최빈값으로 채우거나 KNN 같은 알고리즘을 써서 예측값으로 채움.
2. **중복값 (Duplicate Value):** 시스템 오류 등으로 인해 동일한 데이터가 2번 이상 중복 기록된 상태 
    - 제거: 중복된 행을 찾아 하나만 남기고 삭제함. 제거하지 않으면 특정 데이터의 영향력이 과도하게 커져 모델이 왜곡됨. 
3. **이상치 (Outlier):** 일반적인 데이터 분포에서 지나치게 벗어난 아주 작거나 큰 값
   - 탐지: IQR(사분위수 범위) 규칙(Box plot 활용)이나 Z-Score(표준점수)를 통해 찾아냄.
   - 처리: 단순 입력 오류라면 삭제하거나 정상 범위의 최댓값/최솟값으로 보정(Clipping)함. 단, 사기 거래 감지처럼 이상치 자체가 의미 있는 경우엔 별도로 분리해 학습함. 

### 비유: 요리하기 전 재료 손질하기

- 결측값: 상자 안에 몇몇 사과가 썩어서 속이 텅 비어있다면, 그 사과를 버리거나(삭제), 겉모습이 비슷한 다른 사과로 메꿔 넣는(대치) 과정
- 중복값: 레시피 카드에 '설탕 1큰술'이 인쇄 오류로 두 번 적혀 있어서, 음식을 망치지 않기 위해 한 줄을 지우는 과정
- 이상치: 감자 더미 속에 난데없이 사람 머리만 한 거대 감자(입력 오류)가 들어가 있거나, 감자가 아닌 돌멩이가 섞여 있는 상태. 요리의 일관성을 위해 이를 잘라내거나 골라내야 함.
